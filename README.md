## Description
This LMS (Learning Management System) aims to make coaching and mentoring services easier and more accessible.  
Check the [MVP](/docs/mvp.md) feature list for more information and to stay updated on the progress 
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
